# SOC W06 — 분석가의 눈: 흩어진 경보를 ATT&CK으로 읽고 한 캠페인으로 엮기

> SOC 관제 트랙 6주차. 선행: W01–W05. 인프라: el34. 플랫폼: tw2.

---

## 1. 이번 주의 통찰 — 경보를 "단계"로 읽는다

W05에서 폭주를 줄였다. 남은 경보를 어떻게 읽나? 능숙한 분석가는 경보를 **MITRE ATT&CK** 의 전술/기술로
매핑해 "공격이 지금 **어느 단계**까지 왔나"를 본다. 흩어진 경보가 한 공격자의 **캠페인**으로 보인다.

```
 흩어진 경보               ATT&CK 매핑                  캠페인
 ──────────              ───────────                 ──────
 스캔 경보      →  TA0043 정찰 / T1046           ┐
 SQLi 경보      →  TA0001 초기침투 / T1190        ├→ 한 공격자, 정찰→침투→… 단계
 인증 실패      →  TA0006 자격증명 / T1110        ┘   (출발지·시간으로 상관)
```

---

## 2. ATT&CK 매트릭스 — 전술(Tactic)과 기술(Technique)

- **전술(Tactic)**: 공격의 목적/단계 (정찰 → 초기침투 → 실행 → 지속성 → … → 유출).
- **기술(Technique)**: 그 목적을 이루는 방법 (T번호).

| 경보 | 전술 | 기술 |
|------|------|------|
| 포트 스캔 | Reconnaissance(TA0043) | T1046 Network Service Scanning |
| SQLi | Initial Access(TA0001) | T1190 Exploit Public-Facing App |
| SSH 무차별 대입 | Credential Access(TA0006) | T1110 Brute Force |
| 백도어 계정 | Persistence(TA0003) | T1136 Create Account |
| 인코딩 셸 | Execution(TA0002) | T1059 Command Interpreter |

매핑하면 "정찰만 있다(초기)" vs "이미 지속성까지 갔다(심각)"가 한눈에.

---

## 3. 캠페인 재구성 — 시간순 + 출발지 상관

흩어진 경보를 **시간순**으로 정렬하고 **출발지**로 묶으면 공격 체인이 보인다.
```bash
docker exec el34-siem sh -c 'tail -2000 /var/ossec/logs/alerts/alerts.json | jq -rc "select(.data.src_ip==\"10.20.30.202\")|[.timestamp,.rule.description]|@tsv" | tail -10'
```
- 같은 출발지 + 시간순 = 한 공격자의 단계적 진행.
- ATT&CK 전술 순서(정찰→침투→…)와 시간순이 맞으면 강한 캠페인 신호.

---

## 4. 심층 분석 — ModSec + Suricata

캠페인의 각 단계를 깊게:
- **ModSec audit**: 어떤 페이로드가 어떤 CRS rule(942/941)에, anomaly score 누적이 얼마나(949110).
- **Suricata eve**: signature_id, 출발지/목적지, flow(세션 규모) — 네트워크 단계.

두 심층 분석이 ATT&CK 단계의 증거가 된다.

---

## 5. 에스컬레이션 — P1~P4 판정

분석을 종합해 사건의 우선순위와 에스컬레이션(L2/IR로 올림) 여부를 판정한다.

| 우선순위 | 기준 |
|----------|------|
| **P1** | 침투 성공/지속성 확보/고가치 자산 (즉시 IR) |
| **P2** | 진행형 침투 시도(체인 진전) |
| **P3** | 단발 공격/정찰 |
| **P4** | 정보성/오탐 |

ATT&CK 단계가 깊을수록(정찰<침투<지속성) 우선순위가 높다. 체인이 진전 중이면 에스컬레이션.

## 6. 실습(lab) 형식 — 8 미션

1. **점검**: 텔레메트리 소스
2. **공격 체인 재현**: 스캔→웹→(인증)
3. **ATT&CK 매핑**: 각 경보 → 전술/기술
4. **캠페인 재구성**: 시간순 + 출발지 상관
5. **ModSec 심층**: rule/anomaly score
6. **Suricata 심층**: signature/flow
7. **에스컬레이션 판정**: P1~P4
8. **캠페인 분석 보고서**

> 명령은 el34 호스트(`ssh ccc@192.168.0.151`)에서. 관측·분석 중심.
