# SOC W15 — 기말: APT 캠페인 전체를 SOC 전 역량으로 분석·대응하기 (수료)

> SOC 관제 트랙 15주차 (수료 시험). 선행: W01–W14 전부. 인프라: el34. 플랫폼: tw2.

---

## 1. 수료 시험 — SOC 분석가의 종합 능력

지난 14주의 SOC 역량 — 트리아지(W01), 인증/웹/네트워크 분석(W02-03), 룰 작성(W04), 경보 관리(W05),
ATT&CK 캠페인(W06), SIGMA(W07), 교차 분석(W08), IR(W09), 웹쉘/악성코드/내부위협(W10-12), 인텔(W13),
자동화(W14) — 를 **한 APT 캠페인**에 총동원한다.

```
 APT 캠페인 (다단계)
   분석 → 트리아지 → 교차 상관 → ATT&CK 서사 → IR(격리·제거) → 인텔 격상 → 자동화 → 보고
   (W01-08)        (W05)      (W06,08)      (W06)         (W09-12)      (W13)     (W14)   (전체)
```

핵심 평가: 흩어진 텔레메트리에서 **한 APT 서사를 읽고, 우선순위를 매기고, 대응을 끝까지 수행**하는 능력.

---

## 2. SOC 관점의 APT 대응 흐름

1. **탐지·트리아지**: 다단계 경보를 4요소로 분류, 우선순위.
2. **교차 분석·상관**: 네 소스를 출처/시간으로 한 캠페인 서사로.
3. **ATT&CK 매핑**: 정찰→침투→발판→C2 단계 식별.
4. **IR**: 격리(Contain)→제거(Eradicate)→복구(Recover).
5. **인텔·자동화**: IOC 격상 + 자동 대응으로 재발 방지.
6. **보고**: APT IR 보고서(분석가 종합).

---

## 3. APT 5단계 ↔ SOC 역량

| APT 단계 | SOC 분석 | SOC 대응 |
|----------|----------|----------|
| 정찰 | 스캔 경보 트리아지(W01) | IDS 탐지 |
| 침투 | 웹 교차 분석(W03) | WAF 차단 |
| 발판 | osquery 헌팅(W10-11) | 격리·제거 |
| C2 | IOC 분석(W11) | C2 차단 |
| (전체) | ATT&CK 서사(W06) + 인텔(W13) | IR + 자동화(W14) |

---

## 4. 실습(기말 lab) 형식 — 11 미션

1. **점검**: SOC 전 도구
2. **APT 다단계 재현**: 스캔→웹→인증→호스트 발판
3. **트리아지**: 4요소 + 우선순위
4. **교차 분석**: 네 소스
5. **ATT&CK 서사**: 단계별 매핑
6. **IR Contain**: 출발지 격리(sid 9515001)
7. **IR Eradicate**: osquery 발판 제거
8. **인텔 격상**: IOC 룰(id 101510)
9. **SIEM 수렴 + 출처 상관**
10. **APT SOC 보고서**
11. **정리 확인**: 실습 흔적 0

> 명령은 el34 호스트(`ssh ccc@192.168.0.151`)에서. 룰/발판은 self-clean(공유 인프라 보존).

---

## 5. 수료를 축하한다

15주 동안 너는 경보를 **트리아지하고, 로그를 교차 분석하고, 캠페인을 ATT&CK으로 읽고, 탐지를
작성하고, 사고에 IR로 대응하고, 인텔로 가속하고, 자동화로 24/7 지키는** SOC 분석가의 전 역량을
익혔다. 이제 너는 한 APT 캠페인을 끝까지 분석·대응하는 **SOC 분석가**다. 🎓
