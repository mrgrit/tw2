## 1. 수료 시험 — 한 번의 완전한 침투

W01–W14의 모든 기법(정찰·웹익스플로잇·privesc·persistence·우회·에뮬레이션)을 **PTES 방법론**에 따라
하나의 침투로 완주한다. 방어자는 이 전체 킬체인을 로그로 재구성해 어느 단계서든 끊는다.

```
 PTES 7단계
 ① 사전협의(RoE) ② 정보수집(정찰) ③ 위협모델링 ④ 취약점분석
 ⑤ 익스플로잇(침투→셸→privesc) ⑥ 포스트익스플로잇(persistence→유출) ⑦ 보고
```

## 3. 방어자 — 킬체인 재구성

공격의 각 단계가 방어층에 남긴 흔적을 한 킬체인으로:
```
 정찰 → IDS scan / WAF 913
 침투 → WAF 942/403 + access.log
 셸/발판 → osquery(포트/계정)
 privesc → osquery suid_bin / FIM(sudoers)
 유출 → 네트워크 + FIM(staging)
```
어느 한 단계라도 탐지·차단하면 킬체인이 끊긴다 — defense in depth.

## 4. 실습(캡스톤 lab) 형식 — 10 미션

1. **점검**: 전 도구
2. **정찰(PTES)**: nmap + ffuf
3. **취약점 분석**: 웹 익스플로잇 + 강제 브라우징
4. **익스플로잇/셸**: 셸 캡처(포트 51515)
5. **권한 상승**: SUID/sudo 열거
6. **persistence**: 백도어 계정(pt15ghost)
7. **유출**: 데이터 staging
8. **방어 킬체인 재구성**: 전 단계 흔적
9. **PTES 보고서**
10. **정리**: 전 흔적 self-clean
